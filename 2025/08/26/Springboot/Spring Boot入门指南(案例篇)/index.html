Spring BootжҳҜдёҖдёӘејҖжәҗзҡ„JavaеҹәзЎҖжЎҶжһ¶пјҢе®ғдҪҝеҫ—еҲӣе»әзӢ¬з«Ӣзҡ„гҖҒз”ҹдә§зә§еҲ«зҡ„Springеә”з”ЁеҸҳеҫ—жӣҙе®№жҳ“гҖӮе®ғвҖңи·‘иө·жқҘвҖқеҚіеҸҜз”ЁпјҢеҶ…еөҢдәҶTomcatгҖҒJettyзӯүServletе®№еҷЁпјҢж— йңҖйғЁзҪІWARж–Үд»¶пјҢд№ҹж— йңҖеҚ•зӢ¬зҡ„Servletе®№еҷЁгҖӮ

зҺҜеўғеҮҶеӨҮ

еңЁејҖе§Ӣд№ӢеүҚпјҢиҜ·зЎ®дҝқдҪ зҡ„ејҖеҸ‘зҺҜеўғдёӯе®үиЈ…дәҶд»ҘдёӢиҪҜд»¶пјҡ

  • Java Development Kit (JDK) 8 жҲ–жӣҙй«ҳзүҲжң¬
  • Maven 3.0 жҲ–жӣҙй«ҳзүҲжң¬
  • дёҖдёӘж–Үжң¬зј–иҫ‘еҷЁжҲ–IDEпјҲеҰӮIntelliJ IDEAжҲ–Eclipseпјү
  • Git

еҲӣе»әSpring BootйЎ№зӣ®

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮSpring Initializrеҝ«йҖҹз”ҹжҲҗдёҖдёӘSpring BootйЎ№зӣ®зҡ„еҹәзЎҖз»“жһ„гҖӮ

  1. и®ҝй—® Spring Initializr
  2. йҖүжӢ©з”ҹжҲҗMavenйЎ№зӣ®пјҢйҖүжӢ©JavaиҜӯиЁҖ
  3. жҢҮе®ҡйЎ№зӣ®е…ғж•°жҚ®пјҲGroup, Artifact, Name, Descriptionпјү
  4. ж·»еҠ дҫқиө–пјҲDependenciesпјүпјҢжҲ‘д»¬иҮіе°‘йңҖиҰҒSpring Webдҫқиө–
  5. зӮ№еҮ»вҖңGenerateвҖқз”ҹжҲҗйЎ№зӣ®пјҢдёӢиҪҪе№¶и§ЈеҺӢ

йЎ№зӣ®жһ¶жһ„еӣҫ

еӣҫеұ•зӨәдәҶSpring Bootеә”з”Ёзҡ„еҹәжң¬жһ¶жһ„пјҢеҢ…жӢ¬еҗҜеҠЁзұ»гҖҒжҺ§еҲ¶еҷЁгҖҒжңҚеҠЎгҖҒд»“еә“е’Ңе®һдҪ“зұ»пјҢд»ҘеҸҠе®ғд»¬д№Ӣй—ҙзҡ„е…ізі»гҖӮ

зј–еҶҷHello Worldеә”з”Ё

1. еҗҜеҠЁзұ»

3. иҝҗиЎҢеә”з”Ё

6. иҝҗиЎҢеә”з”Ё
