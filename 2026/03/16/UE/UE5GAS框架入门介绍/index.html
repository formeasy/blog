еүҚиЁҖ

GAS(Gameplay Ability System)жҳҜUE5дёӯз”ЁдәҺеҝ«йҖҹжҗӯе»әMOBA(жҜ”еҰӮиӢұйӣ„иҒ”зӣҹ)гҖҒFPSпјҲжҜ”еҰӮз©ҝи¶ҠзҒ«зәҝгҖҒCSпјүзӯүжёёжҲҸжҠҖиғҪзі»з»ҹзҡ„ејәеӨ§жЎҶжһ¶гҖӮ

дёәд»Җд№ҲйңҖиҰҒGASпјҹ

еҰӮжһңдҪ е·Із»ҸиғҪз”ЁUE5зҡ„и“қеӣҫе’ҢеҠЁз”»зі»з»ҹеҒҡеҮәдёҖдёӘиғҪи·‘иғҪи·ігҖҒдјҡж”ҫеҮ дёӘз®ҖеҚ•жҠҖиғҪзҡ„и§’иүІпјҢйӮЈдҪ иӮҜе®ҡйҒҮеҲ°иҝҮдёӢйқўиҝҷдәӣвҖңзіҹеҝғдәӢе„ҝвҖқгҖӮ

  • жҜ”еҰӮпјҢжғіз»ҷи§’иүІеҠ дёӘвҖңз”ҹе‘ҪеҖјвҖқеұһжҖ§пјҢз»“жһңеҸ‘зҺ°иҝҷдёӘеҖјиҰҒеңЁUIдёҠжҳҫзӨәгҖҒиҰҒиў«жҠҖиғҪдјӨе®іжүЈйҷӨгҖҒиҝҳиҰҒиў«жІ»з–—жҠҖиғҪжҒўеӨҚпјҢдёҚеҫ—дёҚеңЁи“қеӣҫзҡ„еҗ„дёӘи§’иҗҪйҮҢеҶҷдёҠдёҖе ҶвҖңSet HealthвҖқгҖҒвҖңGet HealthвҖқзҡ„иҠӮзӮ№пјҢзЁҚеҫ®ж”№зӮ№йҖ»иҫ‘е°ұеҫ—ж»Ўдё–з•ҢжүҫгҖӮ
  • еҶҚжҜ”еҰӮпјҢжғіеҒҡдёӘвҖңзҒ«зҗғжңҜвҖқжҠҖиғҪпјҢиҰҒжңүж–Ҫжі•еҠЁдҪңгҖҒйЈһиЎҢиҪЁиҝ№гҖҒзў°ж’һдјӨе®іпјҢиҝҳеҫ—жңүдёӘеҶ·еҚҙж—¶й—ҙпјҢеҸҜиғҪеҫ—з”ЁдёҖдёӘзҠ¶жҖҒжңәгҖҒдёҖе Ҷе®ҡж—¶еҷЁгҖҒеҶҚеҠ дёҠдәӢд»¶еҲҶеҸ‘еҷЁжүҚиғҪеӢүејәжӢјеҮәжқҘпјҢд»Јз ҒпјҲжҲ–иҖ…и“қеӣҫпјүд№ұеҫ—еғҸдёҖеӣўйә»гҖӮ

еҲҡејҖе§ӢејҖеҸ‘зҡ„ж—¶еҖҷпјҢйғҪжҳҜйҖҡиҝҮеҹәзұ»зЎ¬еҶҷзҡ„гҖӮеҰӮжһңйЎ№зӣ®йҮҢзҡ„жҠҖиғҪж•°йҮҸдёҠдәҶдёӨдҪҚж•°пјҢеҗ„з§ҚзҠ¶жҖҒж•Ҳжһңзә зј еңЁдёҖиө·пјҢжүҚж„ҸиҜҶеҲ°пјҢжҳҜж—¶еҖҷжүҫдёӘвҖңз®Ўе®¶вҖқдәҶгҖӮиҝҷдёӘз®Ўе®¶пјҢе°ұжҳҜGameplayAbilitySystemпјҢжҲ‘д»¬йҖҡеёёеҸ«е®ғGASгҖӮ

еҸҜд»ҘжҠҠGASзҗҶи§ЈдёәжёёжҲҸдёӯжүҖжңүвҖңи§„еҲҷвҖқе’ҢвҖңиғҪеҠӣвҖқзҡ„дёӯеӨ®з®ЎзҗҶзі»з»ҹгҖӮе®ғеё®з®ЎеҘҪи§’иүІзҡ„еұһжҖ§пјҲжҜ”еҰӮз”ҹе‘ҪгҖҒйӯ”жі•гҖҒж”»еҮ»еҠӣпјүпјҢз®ЎеҘҪи§’иүІзҡ„жҠҖиғҪпјҲжҜ”еҰӮжҷ®ж”»гҖҒзҒ«зҗғгҖҒй—ӘзҺ°пјүпјҢиҝҳз®ЎеҘҪжҠҖиғҪеёҰжқҘзҡ„ж•ҲжһңпјҲжҜ”еҰӮдёӯжҜ’жҺүиЎҖгҖҒеҠ йҖҹBuffпјүгҖӮжңҖйҮҚиҰҒзҡ„жҳҜпјҢе®ғеӨ©з”ҹе°ұдёәзҪ‘з»ңеҗҢжӯҘи®ҫи®ЎеҘҪдәҶгҖӮиҝҷж„Ҹе‘ізқҖпјҢиҙ№иҖҒеӨ§еҠІеңЁжң¬ең°и°ғеҘҪзҡ„жҠҖиғҪпјҢеңЁеӨҡдәәиҒ”жңәж—¶пјҢеӨ§жҰӮзҺҮдёҚдјҡеҮәзҺ°вҖңжҲ‘жү“дёӯдәҶдҪ пјҢдҪ еҚҙжІЎдәӢвҖқиҝҷз§ҚзҒөејӮдәӢд»¶гҖӮ

д»Ӣз»ҚеҺҹзҗҶ

GASжЎҶжһ¶дё»иҰҒз”ұд»ҘдёӢеҮ дёӘжЁЎеқ—жһ„жҲҗ

1. AbilitySystemComponent(ASC)пјҡиғҪеҠӣзі»з»ҹзҡ„з»„д»¶пјҢз”ЁдәҺз®ЎзҗҶзҺ©е®¶жӢҘжңүе“ӘдәӣиғҪеҠӣпјҢе№¶йҖҡиҝҮиҜҘз»„д»¶е°қиҜ•дҪҝз”ЁеҜ№еә”иғҪеҠӣ
2. GameplayAbility(GA)пјҡе…·дҪ“зҡ„иғҪеҠӣпјҢжҜҸз§ҚиғҪеҠӣеҜ№еә”дёҖз§ҚжҲ–еӨҡз§Қж•Ҳжһң
3. GameplayAttributeпјҡжҢҒжңүиғҪеҠӣз»„д»¶зҡ„жӢҘжңүиҖ…зҡ„еұһжҖ§пјҢжҜ”еҰӮжңҖеӨ§з”ҹе‘ҪеҖјпјҢз”ҹе‘ҪеҖјпјҢж”»еҮ»еҠӣзӯү
4. AttributeSet(AS)пјҡеңЁC++дёӯдҝ®ж”№пјҢжҳҜGameplayAttributeзҡ„йӣҶеҗҲпјҢж”ҫзҪ®жүҖжңүеұһжҖ§
5. GameplayEffect(GE)пјҡжҠҖиғҪж•ҲжһңпјҢеҸҜд»Ҙз”ұASCжҲ–GAйҮҠж”ҫпјҢйҖҡеёёз”ЁдәҺдёәзҺ©е®¶ж·»еҠ ж ҮзӯҫпјҢжҲ–иҖ…дҝ®ж”№зҺ©е®¶зҡ„еұһжҖ§зӯү
6. GameplayCue(GC)пјҡз”ЁдәҺж’ӯж”ҫзү№ж•ҲпјҢйҹіж•ҲпјҢй•ңеӨҙж‘Үжҷғзӯүи§Ҷи§үж•Ҳжһң
7. AbilityTaskпјҡUEеҶ…зҪ®дәҶи®ёеӨҡд»»еҠЎпјҢеңЁGAдёӯеҸҜд»Ҙз”ЁеҲ°пјҢжҜ”еҰӮз§»еҠЁеҪ“еүҚзҺ©е®¶дҪҚзҪ®пјҢж’ӯж”ҫеҠЁз”»и’ҷеӨӘеҘҮзӯү
8. GameplayTagsпјҡжң¬иә«е№¶дёҚеұһдәҺGASзі»з»ҹпјҢдҪҶжҳҜGASзі»з»ҹеӨ§йҮҸз”ЁеҲ°пјҢжң¬иҙЁжҳҜдёҖдёӘеҸҜд»ҘеҠЁжҖҒеўһеҲ зҡ„ж ҮзӯҫпјҢеҸҜд»Ҙиө·еҲ°зұ»дјјжһҡдёҫзҡ„дҪңз”ЁпјҢеҶ·еҚҙпјҢжқЎд»¶еҲӨж–ӯйғҪеҸҜд»Ҙз”ЁеҲ°

еҮҶеӨҮйҳ¶ж®ө

1.еҲӣе»әC++з¬¬дёүдәәз§°йЎ№зӣ®пјҲдёҚиғҪз”ЁзәҜи“қеӣҫпјү
GASзҡ„ж ёеҝғеҠҹиғҪжҳҜйҖҡиҝҮ C++ зұ»жҡҙйңІз»ҷи“қеӣҫзҡ„пјҢзәҜи“қеӣҫйЎ№зӣ®ж— жі•дҪҝз”ЁGASзҡ„е…ЁйғЁеЁҒеҠӣпјҲе°Өе…¶жҳҜзҪ‘з»ңйғЁеҲҶпјүгҖӮеҰӮжһңзҺ°еңЁиҝҳжҳҜзәҜи“қеӣҫйЎ№зӣ®пјҢе»әи®®з«ӢеҲ»жҠҠе®ғеҚҮзә§дёә C++ йЎ№зӣ®пјҢеҫҲз®ҖеҚ•пјҢеңЁзј–иҫ‘еҷЁйҮҢйҖүжӢ©вҖңе·Ҙе…·вҖқ->вҖңж–°е»әC++зұ»вҖҰвҖқпјҢйҡҸдҫҝеҲӣе»әдёҖдёӘзұ»пјҲжҜ”еҰӮMyGameModeпјүпјҢUE5е°ұдјҡиҮӘеҠЁдёәдҪ з”ҹжҲҗжүҖйңҖзҡ„VSйЎ№зӣ®ж–Үд»¶гҖӮ

2.еңЁжҸ’д»¶и®ҫзҪ®дёӯеҗҜз”ЁGameplayAbilitiesжҸ’д»¶
еңЁBuild.csж–Үд»¶дёӯж·»еҠ "GameplayTags",вҖңGameplayTasksвҖқ,вҖңGameplayAbilitiesвҖқ,дёүдёӘжЁЎеқ—дҫқиө–гҖӮ

еңЁйЎ¶йғЁиҸңеҚ•ж ҸжүҫеҲ° вҖңзј–иҫ‘вҖқ -> вҖңжҸ’д»¶вҖқгҖӮдјҡеј№еҮәдёҖдёӘеәһеӨ§зҡ„жҸ’д»¶зӘ—еҸЈгҖӮ
еңЁеҸідёҠи§’зҡ„жҗңзҙўжЎҶйҮҢпјҢиҫ“е…Ҙ вҖңGameplayвҖқгҖӮ

дёүдёӘе…ій”®зҡ„жҸ’д»¶пјҡ

  • Gameplay Abilities
  • Gameplay Tags
  • Gameplay Tasks

зЎ®дҝқиҝҷдёүдёӘжҸ’д»¶ж—Ғиҫ№зҡ„еӨҚйҖүжЎҶйғҪжҳҜе·ІеҗҜз”ЁпјҲжү“еӢҫпјүзҠ¶жҖҒгҖӮеҰӮжһңд№ӢеүҚжІЎеҗҜз”ЁпјҢеӢҫйҖүеҗҺпјҢзј–иҫ‘еҷЁдјҡжҸҗзӨәйңҖиҰҒйҮҚеҗҜгҖӮж”ҫеҝғзӮ№еҮ»йҮҚеҗҜпјҢиҝҷжҳҜж ҮеҮҶж“ҚдҪңгҖӮ

жіЁж„ҸпјҡGameplay TagsпјҲжёёжҲҸж ҮзӯҫпјүжҳҜGASзҡ„вҖңиҜӯиЁҖзі»з»ҹвҖқпјҢжҠҖиғҪгҖҒзҠ¶жҖҒгҖҒж•ҲжһңйғҪз”Ёе®ғжқҘж ҮиҜҶе’ҢжҹҘиҜўпјҢиҮіе…ійҮҚиҰҒгҖӮGameplay TasksеҲҷз”ЁдәҺз®ЎзҗҶдёҖдәӣејӮжӯҘд»»еҠЎ

3.еңЁC++йЎ№зӣ®дёӯж·»еҠ жЁЎеқ—дҫқиө–

жҸ’д»¶еҗҜз”ЁеҸӘжҳҜи®©зј–иҫ‘еҷЁи®ӨиҜҶGASдәҶпјҢиҰҒи®©C++д»Јз ҒиғҪи°ғз”ЁGASзҡ„зұ»пјҢиҝҳйңҖиҰҒеңЁзј–иҜ‘еұӮйқўе»әз«Ӣдҫқиө–е…ізі»гҖӮ

жүҫеҲ°йЎ№зӣ®жәҗд»Јз Ғж–Үд»¶еӨ№гҖӮйҖҡеёёеңЁйЎ№зӣ®ж №зӣ®еҪ•дёӢпјҢжңүдёҖдёӘSourceж–Үд»¶еӨ№пјҢйҮҢйқўжңүйЎ№зӣ®еҗҚзҡ„еӯҗж–Үд»¶еӨ№пјҲдҫӢеҰӮMyGasProjectпјүгҖӮ
еңЁиҝҷдёӘж–Үд»¶еӨ№йҮҢпјҢжүҫеҲ°дёҖдёӘеҗҚдёә [йЎ№зӣ®еҗҚ].Build.cs зҡ„ж–Үд»¶гҖӮжҜ”еҰӮжҲ‘зҡ„йЎ№зӣ®еҸ«GasDemoпјҢйӮЈиҝҷдёӘж–Үд»¶е°ұжҳҜGasDemo.Build.csгҖӮ
з”Ёж–Үжң¬зј–иҫ‘еҷЁпјҲжҲ–VS Codeпјүжү“ејҖе®ғгҖӮдјҡзңӢеҲ°зұ»дјјдёӢйқўзҡ„д»Јз Ғпјҡ

йңҖиҰҒдҝ®ж”№зҡ„жҳҜPublicDependencyModuleNames.AddRangeиҝҷдёҖиЎҢгҖӮжҠҠеҲҡжүҚеңЁжҸ’д»¶зӘ—еҸЈзңӢеҲ°зҡ„дёүдёӘжЁЎеқ—еҗҚеӯ—еҠ иҝӣеҺ»пјҡ

еҲӣе»әж ёеҝғз»„д»¶пјҡASCдёҺAttributeSet

GASжңүдёӨдёӘеҹәзҹіиҲ¬зҡ„з»„д»¶пјҡAbility System Component (ASC) е’Ң Attribute Set (AS)гҖӮеҸҜд»ҘжҠҠASCжғіиұЎжҲҗи§’иүІзҡ„вҖңжҠҖиғҪеӨ§и„‘вҖқпјҢжүҖжңүжҠҖиғҪзҡ„жҝҖжҙ»гҖҒеҶ·еҚҙгҖҒзҠ¶жҖҒз®ЎзҗҶйғҪз”ұе®ғиҙҹиҙЈгҖӮиҖҢAttribute SetеҲҷжҳҜи§’иүІзҡ„вҖңеұһжҖ§д»“еә“вҖқпјҢдё“й—Ёеӯҳж”ҫз”ҹе‘ҪеҖјгҖҒйӯ”жі•еҖјгҖҒеҠӣйҮҸгҖҒж•ҸжҚ·иҝҷдәӣж•°еҖјгҖӮ

4.1 еҲӣе»әиҮӘе®ҡд№үзҡ„ASCпјҲеҸҜйҖүдҪҶжҺЁиҚҗпјү

иҷҪз„¶еҸҜд»ҘзӣҙжҺҘжҠҠеј•ж“ҺиҮӘеёҰзҡ„UAbilitySystemComponentжҢӮеҲ°и§’иүІдёҠпјҢдҪҶжҲ‘ејәзғҲе»әи®®и§’иүІеҲӣе»әдёҖдёӘеӯҗзұ»гҖӮиҝҷж ·жңӘжқҘеҸҜд»ҘеҫҲж–№дҫҝең°ж·»еҠ дёҖдәӣиҮӘе®ҡд№үйҖ»иҫ‘пјҢжҜ”еҰӮи®°еҪ•и§’иүІжӢҘжңүзҡ„жҠҖиғҪеҲ—иЎЁпјҢжҲ–иҖ…иҰҶеҶҷдёҖдәӣзҪ‘з»ңеҗҢжӯҘзҡ„еӣһи°ғеҮҪж•°гҖӮ

еңЁеҶ…е®№жөҸи§ҲеҷЁзҡ„ C++ зұ»ж–Үд»¶еӨ№дёҠеҸій”®пјҢйҖүжӢ©вҖңж–°е»әC++зұ»вҖқгҖӮеңЁжҗңзҙўжЎҶйҮҢиҫ“е…ҘAbilitySystemComponentпјҢйҖүжӢ©е®ғдҪңдёәзҲ¶зұ»пјҢз»ҷзұ»иө·дёӘеҗҚеӯ—пјҢжҜ”еҰӮMyAbilitySystemComponentгҖӮеҲӣе»әе®ҢжҲҗеҗҺпјҢжҡӮж—¶дёҚйңҖиҰҒеңЁйҮҢйқўеҶҷд»»дҪ•д»Јз ҒпјҢе®ғд»ЈиЎЁдәҶвҖңиҝҷжҳҜжҲ‘зҡ„жҠҖиғҪзі»з»ҹз»„д»¶вҖқгҖӮ

4.2 еҲӣе»әеұһжҖ§йӣҶпјҲAttribute Setпјү

еұһжҖ§йӣҶжҳҜе®ҡд№үжёёжҲҸж•°еҖјзҡ„ең°ж–№гҖӮжҜ”еҰӮеҲӣе»әдёҖдёӘжңҖеҹәзЎҖзҡ„пјҢеҸӘеҢ…еҗ«з”ҹе‘ҪеҖје’ҢжңҖеӨ§з”ҹе‘ҪеҖјгҖӮ

еҗҢж ·пјҢеҸій”®ж–°е»әC++зұ»пјҢиҝҷж¬ЎзҲ¶зұ»йҖүжӢ©AttributeSetпјҢзұ»еҗҚеҸҜд»ҘеҸ«MyAttributeSetгҖӮ

жү“ејҖз”ҹжҲҗзҡ„MyAttributeSet.hеӨҙж–Үд»¶пјҢжҲ‘д»¬йңҖиҰҒеңЁйҮҢйқўе®ҡд№үеұһжҖ§гҖӮGASзҡ„еұһжҖ§дёҚжҳҜжҷ®йҖҡзҡ„floatеҸҳйҮҸпјҢиҖҢжҳҜз”Ёзү№ж®Ҡзҡ„е®ҸUPROPERTY(BlueprintReadOnly, ReplicatedUsing = OnRep_Health, Category = вҖңAttributeвҖқ)жқҘеЈ°жҳҺпјҢе№¶дё”йңҖиҰҒй…ҚеҘ—зҡ„Getter/SetterеҮҪж•°е’ҢеӨҚеҲ¶еӣһи°ғеҮҪж•°гҖӮ

дёӢйқўжҳҜдёҖдёӘе®ҡд№үдәҶHealthпјҲеҪ“еүҚз”ҹе‘ҪеҖјпјүе’ҢMaxHealthпјҲжңҖеӨ§з”ҹе‘ҪеҖјпјүзҡ„еұһжҖ§йӣҶеӨҙж–Үд»¶зӨәдҫӢпјҡ

еңЁеҜ№еә”зҡ„.cppж–Үд»¶дёӯпјҢйңҖиҰҒе®һзҺ°иҝҷдәӣеҮҪж•°пјҢзү№еҲ«жҳҜзҪ‘з»ңеӨҚеҲ¶йғЁеҲҶпјҡ

е…ій”®зӮ№пјҡеұһжҖ§з”ЁFGameplayAttributeDataзұ»еһӢеЈ°жҳҺпјҢе№¶з”ЁATTRIBUTE_ACCESSORSе®ҸжқҘз”ҹжҲҗдҫҝжҚ·еҮҪж•°пјҢеҗҢж—¶еҝ…йЎ»еӨ„зҗҶеҘҪзҪ‘з»ңеӨҚеҲ¶(ReplicatedUsingе’ҢGetLifetimeReplicatedProps)гҖӮ

4.3 е°Ҷз»„д»¶жҢӮиҪҪеҲ°и§’иүІ

жғідёәзҺ©е®¶ж·»еҠ ASCз»„д»¶пјҢйңҖиҰҒдҝ®ж”№и§’иүІзұ»пјҲйҖҡеёёжҳҜз»§жүҝиҮӘACharacterзҡ„C++зұ»пјүпјҢйҷӨдәҶдёәе®ғж·»еҠ дёҖдёӘUPROPERTYж Үи®°зҡ„и“қеӣҫеҸҜи§Ғзҡ„UAbilitySystemComponentеұһжҖ§еӨ–пјҢиҝҳйңҖиҰҒи®©иҜҘзұ»з»§жүҝиҮӘIAbilitySystemInterfaceжҺҘеҸЈе№¶е®һзҺ°еҜ№еә”жҺҘеҸЈгҖӮ
жү“ејҖи§’иүІзұ»еӨҙж–Үд»¶пјҲдҫӢеҰӮMyCharacter.hпјүпјҢж·»еҠ д»ҘдёӢд»Јз Ғпјҡ

еңЁи§’иүІзұ»зҡ„е®һзҺ°ж–Үд»¶(.cpp)дёӯпјҢйңҖиҰҒеңЁжһ„йҖ еҮҪж•°жҲ–BeginPlayдёӯеҲӣе»әе№¶еҲқе§ӢеҢ–иҝҷдәӣз»„д»¶пјҡ

иҮіжӯӨпјҢи§’иүІе·Із»ҸжӢҘжңүдәҶGASзҡ„ж ёеҝғйӘЁжһ¶гҖӮзј–иҜ‘йЎ№зӣ®пјҢеҰӮжһңжІЎжңүй”ҷиҜҜпјҢжңҖеҹәзЎҖд№ҹжҳҜжңҖз№Ғзҗҗзҡ„й…ҚзҪ®е·ҘдҪңе·Із»Ҹе®ҢжҲҗдәҶпјҒ

е®һзҺ°з¬¬дёҖдёӘжҠҖиғҪпјҡGameplayAbility

жҠҖиғҪпјҲGameplayAbilityпјҢз®Җз§°GAпјүжҳҜGASдёӯд»ЈиЎЁдёҖдёӘеҸҜжҝҖжҙ»иғҪеҠӣзҡ„еҚ•дҪҚгҖӮдёҖдёӘзҒ«зҗғжңҜгҖҒдёҖж¬Ўи·іи·ғгҖҒдёҖж¬Ўж јжҢЎпјҢйғҪеҸҜд»ҘжҳҜдёҖдёӘGAгҖӮжҲ‘д»¬жқҘеҲӣе»әдёҖдёӘжңҖз®ҖеҚ•зҡ„зҒ«зҗғжңҜGAпјҢе®ғйңҖиҰҒеҒҡеҮ д»¶дәӢпјҡжЈҖжҹҘиғҪеҗҰйҮҠж”ҫпјҲжҜ”еҰӮйӯ”жі•еҖјеӨҹдёҚеӨҹпјүгҖҒж’ӯж”ҫж–Ҫжі•еҠЁз”»гҖҒз”ҹжҲҗдёҖдёӘжҠ•е°„зү©гҖҒиҝӣе…ҘеҶ·еҚҙгҖӮ

1 еҲӣе»әGameplayAbilityзұ»

еҸій”®ж–°е»ә C++ зұ»пјҢзҲ¶зұ»йҖүжӢ©GameplayAbilityпјҢиө·еҗҚдҫӢеҰӮGA_FireboltгҖӮGASзҡ„GAзұ»йҖҡеёёд»ҘGA_еүҚзјҖе‘ҪеҗҚпјҢж–№дҫҝз®ЎзҗҶгҖӮ

жү“ејҖGA_Firebolt.hпјҢдјҡзңӢеҲ°е®ғз»§жүҝиҮӘUGameplayAbilityгҖӮжҡӮж—¶дёҚйңҖиҰҒж·»еҠ ж–°зҡ„еҸҳйҮҸпјҢдҪҶйңҖиҰҒдәҶи§ЈеҮ дёӘе…ій”®зҡ„еҸҜйҮҚеҶҷеҮҪж•°пјҡ

  • CanActivateAbilityпјҡжЈҖжҹҘжҠҖиғҪжҳҜеҗҰеҸҜд»ҘйҮҠж”ҫгҖӮиҝҷйҮҢеҸҜд»ҘжЈҖжҹҘи§’иүІжҳҜеҗҰжӯ»дәЎгҖҒжҳҜеҗҰеӨ„дәҺзң©жҷ•зҠ¶жҖҒзӯүгҖӮ
  • ActivateAbilityпјҡжҠҖиғҪжҝҖжҙ»ж—¶и°ғз”Ёзҡ„дё»еҮҪж•°гҖӮеңЁиҝҷйҮҢзј–еҶҷжҠҖиғҪзҡ„дё»иҰҒйҖ»иҫ‘гҖӮ
  • EndAbilityпјҡжҠҖиғҪз»“жқҹж—¶и°ғз”ЁпјҲж— и®әжҲҗеҠҹгҖҒеҸ–ж¶ҲиҝҳжҳҜеӨұиҙҘпјүгҖӮ
  • InputPressed / InputReleasedпјҡеӨ„зҗҶжҠҖиғҪжҢүй”®иҫ“е…ҘгҖӮ

еҜ№дәҺзҒ«зҗғжңҜжҠҖиғҪпјҢдё»иҰҒе…іжіЁActivateAbilityгҖӮжғіи®©е®ғеңЁжңҚеҠЎеҷЁдёҠжү§иЎҢж ёеҝғйҖ»иҫ‘пјҲз”ҹжҲҗжҠ•е°„зү©пјүпјҢеңЁе®ўжҲ·з«ҜдёҠж’ӯж”ҫеҠЁз”»е’Ңйҹіж•ҲпјҲиЎЁзҺ°пјүгҖӮ

2 зј–еҶҷзҒ«зҗғжңҜйҖ»иҫ‘пјҲ C++йғЁеҲҶпјү

е…ҲеңЁеӨҙж–Үд»¶йҮҢеЈ°жҳҺдёҖдёӘUFUNCTIONпјҢз”ЁдәҺеңЁжңҚеҠЎеҷЁдёҠз”ҹжҲҗзҒ«зҗғжҠ•е°„зү©гҖӮ

еңЁ.cppж–Үд»¶дёӯпјҢе®һзҺ°е®ғгҖӮдёәдәҶз®ҖеҢ–пјҢеҒҮи®ҫе·Із»ҸжңүдёҖдёӘи“қеӣҫзұ»BP_FireboltProjectileпјҢе®ғз»§жүҝиҮӘActorжҲ–Projectileз§»еҠЁз»„д»¶пјҢиҙҹиҙЈйЈһеҮәеҺ»е№¶жЈҖжөӢзў°ж’һгҖӮ

д»ҘдёҠд»Јз ҒжҳҜдёҖдёӘй«ҳеәҰз®ҖеҢ–зҡ„зӨәдҫӢпјҢйҮҚзӮ№еұ•зӨәдәҶGAзҡ„з»“жһ„пјҡеңЁActivateAbilityдёӯжЈҖжҹҘжқЎд»¶гҖҒж’ӯж”ҫеҠЁз”»гҖҒйҖҡиҝҮд»»еҠЎзӯүеҫ…еҠЁз”»е®ҢжҲҗгҖҒз„¶еҗҺеңЁжңҚеҠЎеҷЁдёҠз”ҹжҲҗжҠ•е°„зү©гҖӮеңЁе®һйҷ…йЎ№зӣ®дёӯпјҢBP_FireboltProjectileClassйңҖиҰҒйҖҡиҝҮUPROPERTY(EditDefaultsOnly)еңЁзј–иҫ‘еҷЁдёӯжҢҮе®ҡпјҢеҠЁз”»и’ҷеӨӘеҘҮд№ҹйңҖиҰҒеҰҘе–„еј•з”ЁгҖӮ

3 еңЁи“қеӣҫдёӯй…ҚзҪ®жҠҖиғҪе№¶иөӢдәҲи§’иүІ

C++ зұ»еҶҷеҘҪеҗҺпјҢзј–иҜ‘йЎ№зӣ®гҖӮз„¶еҗҺеңЁеҶ…е®№жөҸи§ҲеҷЁдёӯеҸій”® -> и“қеӣҫзұ» -> жүҖжңүзұ»дёӯжҗңзҙў GA_FireboltпјҢеҹәдәҺе®ғеҲӣе»әдёҖдёӘи“қеӣҫпјҢдҫӢеҰӮBP_GA_FireboltгҖӮ

еңЁиҝҷдёӘи“қеӣҫдёӯпјҢеҸҜд»ҘеҒҡеҫҲеӨҡеҸҜи§ҶеҢ–й…ҚзҪ®пјҡ

  • еҶ·еҚҙж—¶й—ҙпјҲCooldownпјүпјҡж·»еҠ дёҖдёӘGameplayEffectпјҲеҗҺйқўдјҡи®ІпјүжқҘе®ҡд№үжҠҖиғҪеҶ·еҚҙгҖӮ
  • ж¶ҲиҖ—пјҲCostпјүпјҡж·»еҠ еҸҰдёҖдёӘGameplayEffectжқҘе®ҡд№үйҮҠж”ҫжҠҖиғҪж¶ҲиҖ—зҡ„йӯ”жі•еҖјгҖӮ
  • жҠҖиғҪж ҮзӯҫпјҲAbility Tagsпјүпјҡз»ҷжҠҖиғҪжү“дёҠж ҮзӯҫпјҢжҜ”еҰӮAbility.Spell.FireпјҢж–№дҫҝе…¶д»–зі»з»ҹиҜҶеҲ«гҖӮ

жҺҘдёӢжқҘпјҢйңҖиҰҒжҠҠиҝҷдёӘжҠҖиғҪиөӢдәҲз»ҷи§’иүІгҖӮ
жңүеҮ з§Қж–№ејҸпјҢжңҖз®ҖеҚ•зҡ„жҳҜеңЁи§’иүІзҡ„и“қеӣҫйҮҢпјҢеңЁBeginPlayдәӢд»¶дёӯпјҢи°ғз”Ёе…¶AbilitySystemComponentзҡ„GiveAbilityеҮҪж•°пјҢе°ҶBP_GA_Fireboltзҡ„зұ»иөӢдәҲз»ҷе®ғгҖӮеҗҢж—¶пјҢиҝҳйңҖиҰҒеңЁи§’иүІзҡ„иҫ“е…Ҙи®ҫзҪ®дёӯпјҢз»‘е®ҡдёҖдёӘжҢүй”®пјҲеҰӮйј ж Үе·Ұй”®пјүжқҘи§ҰеҸ‘иҝҷдёӘжҠҖиғҪпјҲйҖҡиҝҮи°ғз”ЁAbilitySystemComponent->TryActivateAbilityByClassпјүгҖӮ

дёәжҠҖиғҪж·»еҠ ж•ҲжһңпјҡGameplayEffectдёҺGameplayCue

жҠҖиғҪж”ҫеҮәеҺ»дәҶпјҢдҪҶе®ғзҺ°еңЁеҸӘжҳҜдёӘвҖңз©әеЈівҖқгҖӮйңҖиҰҒдёӨдёӘдёңиҘҝжқҘи®©е®ғе®Ңж•ҙпјҡGameplayEffect (GE)жқҘе®ҡд№үе®ғзҡ„вҖңи§„еҲҷвҖқпјҲжҜ”еҰӮдјӨе®іеӨҡе°‘гҖҒеҶ·еҚҙеӨҡд№…пјүпјҢе’ҢGameplayCue (GC)жқҘе®ҡд№үе®ғзҡ„вҖңиЎЁзҺ°вҖқпјҲжҜ”еҰӮе‘Ҫдёӯж—¶зҡ„зҲҶзӮёзү№ж•Ҳе’Ңйҹіж•ҲпјүгҖӮ

1 еҲӣе»әеҶ·еҚҙдёҺж¶ҲиҖ—зҡ„GameplayEffect

GameplayEffectжҳҜGASдёӯдҝ®ж”№еұһжҖ§е’Ңеә”з”ЁзҠ¶жҖҒзҡ„ж ёеҝғж•°жҚ®иө„дә§гҖӮе®ғдёҚеҢ…еҗ«йҖ»иҫ‘пјҢеҸӘеҢ…еҗ«й…ҚзҪ®гҖӮдёәзҒ«зҗғжңҜеҲӣе»әдёӨдёӘGEпјҡ

  • еҶ·еҚҙGEпјҡдёҖдёӘDurationпјҲжҢҒз»ӯж—¶й•ҝпјүзұ»еһӢзҡ„GEпјҢжҢҒз»ӯж—¶й—ҙдёә5з§’гҖӮе®ғжң¬иә«дёҚдҝ®ж”№д»»дҪ•еұһжҖ§пјҢдҪҶжӢҘжңүдёҖдёӘCooldownж ҮзӯҫпјҲеҰӮCooldown.FireboltпјүгҖӮеҪ“иҝҷдёӘGEеңЁжҠҖиғҪдёҠжҝҖжҙ»ж—¶пјҢжҠҖиғҪеңЁ5з§’еҶ…ж— жі•еҶҚж¬ЎдҪҝз”ЁгҖӮ
  • ж¶ҲиҖ—GEпјҡдёҖдёӘInstantпјҲзһ¬ж—¶пјүзұ»еһӢзҡ„GEпјҢеңЁжҠҖиғҪйҮҠж”ҫж—¶з«ӢеҚіжүЈйҷӨи§’иүІзҡ„йӯ”жі•еҖјпјҲManaпјүгҖӮиҝҷйңҖиҰҒеңЁAttributeSetдёӯе…Ҳе®ҡд№үеҘҪManaеұһжҖ§гҖӮ

еҲӣе»әGEйқһеёёз®ҖеҚ•пјҢеңЁеҶ…е®№жөҸи§ҲеҷЁдёӯеҸій”® -> и“қеӣҫзұ» -> жүҖжңүзұ»дёӯжҗңзҙў GameplayEffectпјҢз„¶еҗҺйҖүжӢ©InstantжҲ–Durationзұ»еһӢеҲӣе»әеҚіеҸҜгҖӮеңЁGEзҡ„и“қеӣҫдёӯпјҢеҸҜд»Ҙпјҡ

  • еңЁModifiersпјҲдҝ®йҘ°еҷЁпјүеҲ—иЎЁдёӯж·»еҠ дёҖжқЎпјҢйҖүжӢ©AttributeSetдёӯзҡ„ManaеұһжҖ§пјҢи®ҫзҪ®Modifier OpдёәAddпјҲзӣёеҠ пјүпјҢMagnitudeдёә-20.0пјҲиЎЁзӨәжүЈйҷӨ20зӮ№пјүгҖӮ
  • еңЁGranted TagsпјҲжҺҲдәҲж ҮзӯҫпјүжҲ–Gameplay Effect TagsпјҲGEж Үзӯҫпјүдёӯж·»еҠ еҶ·еҚҙж ҮзӯҫгҖӮ

з„¶еҗҺпјҢеӣһеҲ°жҠҖиғҪи“қеӣҫBP_GA_FireboltпјҢеңЁAbility DetailsйқўжқҝдёӯпјҢжүҫеҲ°Cooldownе’ҢCostйҖүйЎ№пјҢеҲҶеҲ«е°ҶеҲӣе»әзҡ„еҶ·еҚҙGEе’Ңж¶ҲиҖ—GEжӢ–иҝӣеҺ»гҖӮиҝҷж ·пјҢGASжЎҶжһ¶е°ұдјҡеңЁжҠҖиғҪжҝҖжҙ»ж—¶иҮӘеҠЁеә”з”Ёиҝҷдәӣж•ҲжһңгҖӮ

2 еҲӣе»әе‘Ҫдёӯзҡ„GameplayCue

GameplayCueз”ЁдәҺеӨ„зҗҶдёҺжёёжҲҸйҖ»иҫ‘ж— е…ізҡ„и§Ҷеҗ¬иЎЁзҺ°гҖӮзҒ«зҗғе‘Ҫдёӯзӣ®ж Үж—¶пјҢжғіж’ӯж”ҫдёҖдёӘзҲҶзӮёзІ’еӯҗе’Ңйҹіж•ҲгҖӮ

йҰ–е…ҲпјҢйңҖиҰҒеҲӣе»әдёҖдёӘGameplayCueйҖҡзҹҘзұ»гҖӮеңЁC++дёӯпјҢж–°е»әдёҖдёӘзұ»з»§жүҝиҮӘUGameplayCueNotify_StaticпјҲдёҖж¬ЎжҖ§ж•ҲжһңпјүжҲ–UGameplayCueNotify_ActorпјҲжҢҒз»ӯж•ҲжһңпјҢйңҖиҰҒйҷ„зқҖеҲ°ActorпјүгҖӮеҜ№дәҺзҲҶзӮёпјҢжҲ‘д»¬з”ЁStaticгҖӮ

еҗҢж ·пјҢзј–иҜ‘еҗҺпјҢеңЁеҶ…е®№жөҸи§ҲеҷЁдёӯеҹәдәҺжӯӨзұ»еҲӣе»әи“қеӣҫBP_GCN_FireboltHitпјҢе№¶еңЁи“қеӣҫдёӯжҢҮе®ҡе…·дҪ“зҡ„зІ’еӯҗзі»з»ҹExplosionParticleе’ҢеЈ°йҹіExplosionSoundгҖӮ

3 еңЁжҠ•е°„зү©дёӯи§ҰеҸ‘GameplayCue

жңҖеҗҺпјҢдҝ®ж”№зҒ«зҗғжҠ•е°„зү©и“қеӣҫBP_FireboltProjectileгҖӮеңЁе…¶зў°ж’һдәӢд»¶пјҲOnHitпјүдёӯпјҢеңЁйҖ жҲҗдјӨе®іпјҲйҖҡиҝҮApplyGameplayEffectToTargetеә”з”ЁдёҖдёӘзһ¬ж—¶дјӨе®іGEпјүзҡ„еҗҢж—¶пјҢжү§иЎҢGameplayCueгҖӮ
